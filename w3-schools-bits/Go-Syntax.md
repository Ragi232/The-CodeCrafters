##  What “Go syntax” really means

Go syntax refers to the set of rules that define how you write valid Go programs so the compiler can understand and execute them. 
Go syntax is how you turn your thoughts into structured, exact instructions that both the computer and other programmers can understand without confusion.Go syntax is the set of rules for writing valid Go programs.

## e.g 

package main
import ("fmt")

func main()
{
  fmt.Println("Hello World!")
}
