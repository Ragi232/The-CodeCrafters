##  Go Comments — Explained Simply
Go comments are notes you write inside your code that the computer completely ignores. They are there to help you and other people understand the code better.Comments can also be used to prevent code execution when testing an code.Go supports single-line or multi-line comments.Go comments are not just extra text—they are a communication tool, a guide for understanding, and a critical aid for maintaining and documenting code effectively.

## e.g

// This is a comment
package main
import ("fmt")

func main() {
  // This is a comment
  fmt.Println("Hello World!")
}
